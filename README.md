bakalari-crawler
================

API a Crawler pro IS Bakaláři. Vrací vstup ve formě JSON. 

#Instalace
Uprav a přejmenuj soubor `config/config_demo.php` na `config/config.php`.
Vyžadováno je `PHP 5.3`.

#API dokumentace
http://docs.skolar.apiary.io/
